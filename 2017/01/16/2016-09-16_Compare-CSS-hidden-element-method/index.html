CSS“隐藏”元素的几种方法的对比

display:none

设置元素的display为none是最常见的隐藏元素的方法。

将元素设置为none后，该元素在页面上将彻底消失，元素原本占用的空间也将消失，就是说它会导致元素重拍和浏览器重绘。

visibility:hidden

设置元素的visibility为hidden也属于常用的隐藏方法。它和display的区别是，仅仅是元素本身隐藏掉，但元素本来占据的空间依旧会保留，所以他只会导致浏览器重绘而不会导致重排。

visibility:hidden适用于那些元素隐藏后不希望页面布局发生变化的场景。

opacity:0

opacity属性表示透明度，此处表示将元素透明度设置为0后，在我们看来元素就相当于隐藏了，可以算是一种隐藏元素的方法。

这种方法和visibility:hidden的效果是一样的，都是隐藏后依然元素空间还在。实际上是因为元素只是透明度为0，但元素还在那，只是看不到了而已，依旧存在页面中。
